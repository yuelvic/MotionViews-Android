# Text Stickers for Android
based on [uptechteam's](https://github.com/uptechteam) [MotionViews-Android](https://github.com/uptechteam/MotionViews-Android)

## Added Features
* Text background
* Restore state after editing

## Removed Features
* Image sticker

## How to use
To add background
```java
TextEntity.setBorder(true)
```
